Solution Differentiators

Architected to work for multiple cloud providers and displays cloud carbon metrics in a single, holistic view

Supports AWS and Google Cloud, with plans to add Azure and other providers, including potential for on-prem and co-located data centers

Ability to measure usage at a granular service level and use actual server utilization rather than average utilization for hyperscale data centers

Provides estimates for both energy and carbon emissions from cloud usage

Displays information in chart visual, table, or CSV

The solution is open and extensible; contributions and improvements encouraged
